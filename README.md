# tfg-superconductors

Código utilizado para el análisis numérico de mi Trabajo de Fin de Grado en Física en la Universitat de València.

**Título:** ¿Superconductores a temperatura ambiente?

**Autor:** Javier Cabrera Torres.

**Tutor:** Daniel Errandonea Ponce.

## Contenido

- `bcs_gap.py`: Resolución numérica de la ecuación del gap BCS a temperatura finita mediante el método de bisección (Python, SciPy).
- `penetracion_campo.m`: Visualización del decaimiento del campo electromagnético en un cable superconductor rectangular (MATLAB).

## Requisitos

- Python 3 con NumPy, SciPy y Matplotlib (para el script de bcs_gap).
- MATLAB (para el script de penetración de campos).
